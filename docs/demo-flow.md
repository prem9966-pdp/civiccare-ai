# CivicCare AI: Hackathon Demo Flow (v1.0.2)
*Total Time: 3-5 Minutes*

---

## 🎭 Persona: Aarav (Common Citizen)
- **Goal**: Aarav is looking for help with medical expenses for his elderly parents and doesn't know where to start.

---

## 🏁 Phase 1: Onboarding (0-60s)
1. **Login**: Use `aarav@example.com` / `password123`.
2. **Dashboard**: Show the premium, glassmorphic UI. Highlight "Recent Alerts" (Notifications) and "Proximity Discovery" (Hospitals).
3. **Profile**: Go to the Profile page. Note how the "Discovery Ready" status is calculated based on occupation (Farmer) and Income (Below 3L).

---

## 🤖 Phase 2: Counselor GPT (60-120s)
1. **AI Chat**: Open the Chat module.
2. **Contextual Query**: Ask: *"I need help with my father's surgery costs."*
3. **AI Response**: Show how the AI suggests **PM-JAY (Ayushman Bharat)** based on Aarav's profile.
4. **Emergency Check**: Type: *"Help, it's a medical emergency!"*. Show the **Emergency Card** popping up with dial buttons and hospital maps.

---

## 📜 Phase 3: Match Engine & Vault (120-180s)
1. **Explore Schemes**: Navigate to the Schemes page. Filter by "Health".
2. **PM-JAY Audit**: Open the PM-JAY detail page. Show the **Why You Qualify** checklist.
3. **Citizen Vault**: Go to the Documents page. Show how Aarav's Aadhar is already "Verified", but he needs to upload an "Income Certificate" to hit 100% readiness.

---

## 📝 Phase 4: Digital Advocacy (180-240s)
1. **Letter Generation**: Go to the Advocacy Letters page.
2. **Drafting**: Input a complaint about "Long wait times at the local health center".
3. **Review & Export**: Show the **AI-generated formal letter** in the premium previewer. Click **Export PDF** to show ready-to-print documentation.

---

## 🛡️ Phase 5: Admin Control Matrix (240-300s)
1. **Admin Login**: Switch to `admin@civiccare.ai` / `adminpassword`.
2. **Global Analytics**: Show total citizens and active schemes.
3. **Regional Load**: Point to the "Emergency Load Monitor" showing health-system pressure in Delhi/Mumbai.
4. **Catalog CRUD**: Briefly show how an admin can add or edit schemes.
